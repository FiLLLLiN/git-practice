## 3. Работа с локальным Git

### Основные команды

git init - создать репозиторий  
пример: git init  

git status - состояние файлов  
пример: git status  

git add . - добавить файлы  
пример: git add .  

git commit -m "сообщение" - сохранить  
пример: git commit -m "init"  

git commit --amend - изменить коммит  
пример: git commit --amend -m "fix"  

git log --oneline - история  
пример: git log --oneline  

git diff - изменения  
пример: git diff  

---

### Работа с ветками

git branch - список веток  
пример: git branch  

git checkout -b new-branch - создать ветку  
пример: git checkout -b test  

git checkout main - перейти  
пример: git checkout main  

git merge test - объединить  
пример: git merge test  

git branch -d test - удалить  
пример: git branch -d test
